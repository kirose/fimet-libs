# FIMET

## fimet-libs 2.2.0

This project is a java libray for ISO8583 tests automation

## Features:

* Use case execution
* Stress execution
* Simulator configuration
* Socket manipulation

## Third-party librares

* **GSON** used for JSON manipulation
