# FIMET

## fimet-libs 1.0.0

This project is a java libray for ISO8583 tests automation

## Features:

* Use case execution
* Stress execution
* Simulator configuration
* Socket manipulation

## Third-party librares

* **GSON** used for JSON manipulation
* **ORMLITE** used for persist field formats, sockets and other information

## Examples

See some examples [here](https://github.com/kirose/fimet-libs/tree/master/fimet-test/src/main/java/com/fimet/test)
